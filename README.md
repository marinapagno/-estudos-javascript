# Estudos de JavaScript

Repositório de aprendizado de JavaScript, parte da minha transição de carreira de jornalismo digital para desenvolvimento web.

Aqui guardo exercícios, anotações e mini-projetos enquanto aprendo do zero. Os mini-projetos são aplicados ao contexto real de uma cliente: um CRM simples para uma advogada trabalhista.
